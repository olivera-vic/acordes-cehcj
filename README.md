# Acordes CEHCJ

Aplicación web progresiva (PWA) desarrollada para la **Comunidad Evangélica de Hermanos en Cristo Jesús (CEHCJ)**.

Permite consultar el repertorio de alabanza desde cualquier dispositivo, con transposición automática de acordes y una interfaz optimizada para teléfonos móviles.

## Características

* Búsqueda rápida de canciones.
* Transposición de acordes (+ / -).
* Visualización optimizada para celulares.
* Instalación como aplicación (PWA).
* Base de datos en Supabase.
* Despliegue automático mediante Vercel.

## Tecnologías utilizadas

* React
* Vite
* React Router
* Bootstrap 5
* Supabase
* Vercel

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/olivera-vic/acordes-cehcj.git
```

Entrar al proyecto:

```bash
cd acordes-cehcj
```

Instalar dependencias:

```bash
npm install
```

Crear un archivo `.env` con las variables:

```env
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```

Ejecutar en modo desarrollo:

```bash
npm run dev
```
